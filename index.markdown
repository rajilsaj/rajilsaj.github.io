## Welcome

This is your new home page, styled with your custom template!

You can now start creating your own content. Here are some next steps:

*   **Create more pages:** Create files like `about.md` and add `layout: default` to the front matter.
*   **Create posts:** Add new markdown files to the `_posts` directory.
*   **Customize your styles:** Edit `assets/css/main.css` to change the look and feel of your site.
*   **Create includes:** For reusable components like a navigation menu, create files in the `_includes` directory.
